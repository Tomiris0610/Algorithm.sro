Ответы на вопросы
 1. Чем очередь отличается от стека?
Очередь работает по принципу FIFO (первый пришёл — первый обслужен).  
Стек работает по принципу LIFO (последний пришёл — первый вышел).
 2. Что означает FIFO?
FIFO (First In, First Out) — первый вошёл, первый вышел.
 3. Где используются очереди в реальной жизни?
- Очереди в банке
- Очереди в магазине
- Обработка задач в операционных системах
- Очередь печати
- Обработка запросов на сервере
